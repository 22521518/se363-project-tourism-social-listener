# Frontend

Dashboard and visualizations for the Social Listening platform.

## Structure

| Folder | Purpose |
|--------|---------|
| `web/streamlit/` | Python Streamlit dashboard (working) |

## Quick Start

```powershell
cd web/streamlit
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Technology

- **Current**: Streamlit + Plotly for real-time ABSA dashboard
- **Planned**: React + Vite SPA
