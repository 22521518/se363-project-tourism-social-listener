# API

The API layer exposes REST endpoints for dashboards and consumers to retrieve mentions, aggregates, and metadata.

## Structure

| Folder | Purpose |
|--------|---------|
| `controllers/` | Request handlers and route definitions |
| `entities/` | Domain entity definitions |
| `repositories/` | Data access layer |
| `services/` | Business logic |

## Technology

- **Framework**: Node.js / Express or Fastify (planned)
- **Contract**: OpenAPI-first design
- **Storage**: PostgreSQL, OpenSearch
